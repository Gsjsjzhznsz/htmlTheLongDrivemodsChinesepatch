# 长途旅行模组安装器

一款为游戏《长途旅行》(The Long Drive) 设计的模组管理工具，提供图形化界面，让您轻松浏览、安装、更新和卸载模组。

---

## 主要功能

### 🔍 模组管理
- **多源支持**：自动从多个数据源获取模组列表，支持故障转移
- **搜索与筛选**：按名称、作者搜索，按分类筛选
- **双视图模式**：卡片视图与列表视图自由切换
- **详情查看**：点击模组卡片查看完整信息（描述、依赖、图片等）

### 📦 安装系统
- **一键安装**：点击按钮即可完成安装
- **依赖自动处理**：自动安装所需的前置模组
- **进度显示**：实时显示下载速度、进度和剩余时间
- **格式支持**：支持 `.dll` 文件和 `.zip` 压缩包
- **批量安装**：通过模组包（Modpack）一键安装预设集合

### 🗑️ 模组管理
- **精确卸载**：仅删除模组的 `.dll` 文件，保留配置文件
- **更新检测**：自动检测是否有新版本可用
- **版本记录**：记录已安装模组的版本信息

### 🎨 用户界面
- **现代设计**：深色主题，圆角卡片，流畅动画
- **响应式布局**：完美适配桌面与移动设备
- **Toast 通知**：操作结果以非侵入式提示呈现
- **下载管理器**：悬浮窗显示所有下载任务，可折叠

---

## 技术架构

### 后端
- **框架**：Python Flask，多线程支持
- **下载管理**：带进度回调的流式下载，支持断点续传
- **文件追踪**：使用 JSON 清单文件记录每个模组安装的文件路径
- **网络请求**：内置重试机制（5次重试，10秒超时）

### 前端
- **纯原生**：HTML/CSS/JavaScript，无外部依赖
- **实时更新**：轮询获取下载任务状态
- **本地存储**：记住用户选择的视图模式和数据源

### 文件结构
```
Documents/TheLongDrive/Mods/
├── temp/
│   └── Versions/
│       ├── {模组名}.txt           # 版本信息
│       └── {模组名}_manifest.json # 安装文件清单
```

## 快速开始

### 环境要求
- Python 3.7 或更高版本
- Flask 库（`pip install flask requests`）

### 安装与运行
1. 下载项目代码
2. 安装依赖：
   ```bash
   pip install flask requests
   ```
3. 启动服务：
   ```bash
   python app.py
   ```
4. 打开浏览器访问 `http://127.0.0.1:5000`

## 使用指南

### 浏览模组
- 在顶部选择数据源（默认官方源）
- 使用搜索框输入名称或作者进行检索
- 通过分类下拉框筛选特定类型
- 点击模组卡片查看详细信息（图片、描述、依赖等）

### 安装模组
- **单个模组**：在浏览页面点击“安装”按钮
- **模组包**：切换到“模组包”标签页，点击“安装模组包”
- 安装任务会出现在右下角的下载管理器中，实时显示进度、速度和剩余时间

### 管理已安装模组
- 切换到“已安装模组”标签页
- 查看所有已安装模组及其版本
- 点击“卸载”删除模组（仅删除 `.dll` 文件）
- 当有新版本时，“更新”按钮会高亮，点击即可升级

### 下载管理器
- 固定在右下角，点击头部可折叠/展开
- 显示每个任务的名称、进度、速度和状态
- 支持多个任务并发下载

## 数据源说明

应用支持多源模组列表，优先级如下：

1. **官方源 (GitLab)** – 社区维护的主源，推荐使用
2. **极狐源 (GitLab China)** – 国内镜像，适合中国用户
3. **GitHub 镜像** – 备用源，确保可用性
4. **本地源** – 离线模式，使用本地 `modlist_3.json` 文件

当首选源不可用时，应用会自动尝试下一个源。

## 常见问题

### Q: 模组列表无法加载？
- 检查网络连接
- 当前源可能暂时不可用，系统会自动切换到备用源
- 确保本地 `modlist_3.json` 存在（用于离线模式）

### Q: 安装失败？
- 确认模组文件存在于服务器
- 检查磁盘空间是否充足
- 确保模组没有与其他已安装模组冲突

### Q: 下载速度慢或卡住？
- 可尝试切换到国内源（极狐源）
- 检查防火墙或代理设置
- 下载管理器会显示速度，请耐心等待

### Q: 卸载后还有残留文件？
- 本工具仅删除模组的 `.dll` 文件，保留配置和资源文件
- 如需彻底清理，可手动删除对应文件夹

## 开发者信息

### 添加新数据源
编辑 `app.py` 中的 `MODLIST_SOURCES` 和 `MODPACK_SOURCES` 数组：

```python
{
    "name": "源名称",
    "url": "https://example.com/modlist.json"
}
```

### API 端点
- `GET /api/mods?source=X` – 获取模组列表
- `GET /api/modpacks?source=X` – 获取模组包列表
- `POST /api/install` – 安装模组
- `POST /api/uninstall` – 卸载模组
- `POST /api/update` – 更新模组
- `GET /api/tasks` – 获取下载任务状态
